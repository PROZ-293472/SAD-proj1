library(dplyr)
df <- df %>% group_by(station_name) %>% mutate(max_temp_diff = max_daily_temp - lag(max_daily_temp))
df <- df %>% group_by(station_name) %>% mutate(mean_temp_diff = mean_daily_temp - lag(mean_daily_temp))

W celu usunicia wartoci NaN, zastpiono je zerami.

df$max_temp_diff[is.na(df$max_temp_diff)] <- 0
df$mean_temp_diff[is.na(df$mean_temp_diff)] <- 0

mean_temps <- c()

for (l in list(loc1, loc2, loc3)){

Zadanie 2

a) Badanie liczby katastrof

Przygotowanie danych dotyczcych wypadkw w kolejnych kwartaach wymagao zakodowania dat w odpowiednim formacie i wyznaczenia na ich podstawie kwartaw.

catastrophes <- read.csv(file = "./data/katastrofy.csv")
catastrophes$Date <- as.Date(catastrophes[["Date"]], format = "%m/%d/%Y")
catastrophes$Quarters <- lubridate::quarter(catastrophes$Date, with_year = TRUE)

Z przygotowanych danych moemy wygenerowa histogram przedstawiajcy jak zmieniaa si czestotliwo zdarze w nastpujcych po sobie kwartaach.

Jak widzimy na powyszym wykresie, liczba katastrof lotniczych zmieniaa si na przestrzeni lat. Ponadto w zwizku z dugim okresem poddanym analizie oraz ze zmianami zachodzcymi w rzeczywistych procesach zwizanych z rozwojem lotnictwa, nie jestemy w stanie modelowa caoci danych jednym rozkadem. Jednakowo moemy wydzieli grupy, pochodzce z krtszych okresw.

cat_subset <- catastrophes[between(catastrophes$Date, as.Date("1940-01-01"), as.Date("1956-01-01")),]

poisson <- as.data.frame(dpois(15:30, lambda = 24))
poisson$x <- 1940:1955
colnames(poisson) <- c("y", "x")

Na poniszym wykresie widzimy, e wypadki pochodzce z okresu 1940-1955. Po dobraniu odpowiedniego parametru lambda=24 dla rozkadu Poissona udao si w przyblieniu zamodelowa wybrany fragment danych.
